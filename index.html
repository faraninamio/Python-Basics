What is Python?

Python is a general-purpose programming language known for its simplicity and readability. It reads almost like plain English, making it one of the best languages to learn first.

📊
Data Science & ML

Power tools like NumPy, Pandas, and TensorFlow are all Python-based.

🌐
Web Development

Frameworks like Django and Flask let you build full web apps.

🤖
Automation

Automate boring tasks — file management, web scraping, scheduling.

🔒
Cybersecurity

Used extensively in penetration testing and security tooling.

🖥️
Embedded Systems

MicroPython runs on Raspberry Pi and microcontrollers.

⬇️
Install Python

Download from python.org — available for Windows, Mac & Linux.

02

Variables

Variables are like labeled boxes — you store a value and refer to it by name later.

# Declare a variable with: name = value
name = 'John Doe'
age  = 25
price = 9.99
is_logged_in = True

Naming Rules

✅ Valid names
Must start with a letter or _
Only letters, numbers, underscores
Use snake_case for multiple words
❌ Invalid names
Cannot start with a number: 2fast
No spaces: my variable
No reserved keywords: if, class
# Case-sensitive — these are 3 different variables!
age = 10
Age = 20
AGE = 30
03

Data Types

Python is dynamically typed — you don't declare types manually. Python figures it out from the value you assign.

TypeWhat it isExample
intWhole number42
floatDecimal number3.14
strText'hello'
boolTrue or FalseTrue
listOrdered, changeable[1, 'hi', 3.14]
tupleOrdered, fixed(7, 5, 8)
dictKey-value pairs{"name": "Alice"}
setUnique items, unordered{1, 2, 3}
NoneNo valueNone
 
print(type('hello'))           # <class 'str'>
print(isinstance('hi', str))  # True
print(isinstance(42, str))   # False

Mutable vs Immutable

Immutable — can't be changed after creation
int, float, str, bool, tuple, None
Mutable — can add, remove, or update items
list, dict, set
04

Working with Strings

Strings are sequences of characters. Use single or double quotes — just stay consistent.

Accessing Characters & Slicing

my_str = 'Hello world'
print(my_str[0])    # H  (first character)
print(my_str[-1])   # d  (last character)
print(my_str[0:5])  # Hello
print(len(my_str))  # 11

f-Strings

name = 'Jessica'
age  = 28
print(f'My name is {name} and I am {age}.')

Common String Methods

s = 'hello world'
s.upper()                     # 'HELLO WORLD'
s.replace('hello', 'hi')      # 'hi world'
s.split(' ')                  # ['hello', 'world']
' '.join(['a', 'b'])          # 'a b'
s.startswith('h')            # True
'world' in s                  # True
05

Working with Numbers

print(56 + 12)    # 68   — addition
print(56 - 12)    # 44   — subtraction
print(56 * 12)    # 672  — multiplication
print(56 / 12)    # 4.67 — division (always float)
print(56 % 12)    # 8    — modulus (remainder)
print(56 // 12)   # 4    — floor division
print(4 ** 2)     # 16   — exponentiation

Useful Built-in Functions

float(4)      # 4.0  — int to float
int(4.9)      # 4    — float to int (truncates)
round(7.7)    # 8    — nearest integer
abs(-13)      # 13   — absolute value
pow(2, 3)     # 8    — 2 to the power of 3

Augmented Assignments

x = 10
x += 5   # 15
x -= 3   # 12
x *= 2   # 24
x /= 4   # 6.0
x **= 2  # 36.0
06

Functions

Functions are reusable blocks of code. Define once, call anytime.

def get_sum(num_1, num_2):
    return num_1 + num_2

result = get_sum(3, 4)
print(result)  # 7

Default Parameter Values

def greet(name, greeting='Hello'):
    return f'{greeting}, {name}!'

print(greet('Alice'))       # Hello, Alice!
print(greet('Bob', 'Hey'))  # Hey, Bob!

No Return Value

def say_hi():
    print('hi')

result = say_hi()   # prints: hi
print(result)       # None
07

Scope

Scope determines where a variable can be accessed in your code.

🏠
Local Scope

Variables inside a function — only accessible within that function.

🪆
Enclosing Scope

An inner function can read variables from its outer function.

🌍
Global Scope

Variables outside all functions — accessible from anywhere.

⚙️
Built-in Scope

Python's predefined names: print, str, type, len, etc.

tax = 0.70  # global

def get_total(subtotal):
    total = subtotal + (subtotal * tax)  # accesses global
    return total

print(get_total(100))  # 170.0

Truthy, Falsy & Booleans

Every value in Python is secretly True or False in a logical context.

❌ Falsy (treated as False)
False00.0 ''None[]{}
✅ Truthy (treated as True)
True13.14 'hello'[1,2,3]{'a':1}

Boolean Operators: and, or, not

# AND — both must be true
if is_citizen and age >= 18:
    print('Can vote')

# OR — at least one must be true
if age < 18 or is_student:
    print('Student discount')

# NOT — flips the value
if not is_admin:
    print('Access denied')

# Short-circuiting
print(False and 10)  # False
print(True or 10)   # True
print(True and 10)  # 10

full_dot  = '●'
empty_dot = '○'

def create_character(name, strength, intelligence, charisma):
    if not isinstance(name, str): return 'The character name should be a string'
    if name == '':               return 'The character should have a name'
    if len(name) > 10:           return 'The character name is too long'
    if ' ' in name:               return 'The character name should not contain spaces'

    stats = [strength, intelligence, charisma]
    if not all(isinstance(s, int) for s in stats): return 'All stats should be integers'
    if any(s < 1 for s in stats):  return 'All stats should be no less than 1'
    if any(s > 4 for s in stats):  return 'All stats should be no more than 4'
    if sum(stats) != 7:            return 'The character should start with 7 points'

    def bar(v): return full_dot * v + empty_dot * (10 - v)
    return f"{name}\nSTR {bar(strength)}\nINT {bar(intelligence)}\nCHA {bar(charisma)}"

print(create_character('ren', 4, 2, 1))
