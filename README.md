# Python Basics

A beginner-friendly guide to Python fundamentals. This project covers essential topics such as:

- Variables and Data Types  
- Strings and Numbers  
- Lists, Tuples, and Dictionaries  
- Functions and Loops  
- Conditional Statements  

## Project Purpose

This site is designed to help beginners practice and understand Python concepts interactively. It’s hosted on GitHub Pages, so you can view it live here: [Python Basics](https://faraninamio.github.io/Python-Basics/)

## Built With

- HTML  
- CSS  
- GitHub Pages for hosting

## How to Use

1. Clone or fork this repository  
2. Open `index.html` in your browser  
3. Explore the examples and practice coding in Python
